# flight-price-dashboard
An interactive data analytics dashboard built with Python, Streamlit, and Plotly to analyze and visualize domestic flight ticket price trends in India. The project processes flight records to explore how factors like airlines, route demand, booking time, and stops affect ticket pricing, providing clear insights through a clean user interface.
 
## Features
- Dynamic filtering options by airlines, source, destination, and travel metrics.
- Interactive data visualizations including price distribution plots, scatter trends, and category charts.
- Clean layout design with a dedicated automated insight engine for data reports.

## Tech Stack
- Python
- Streamlit
- Plotly Express
- Pandas
- NumPy
